Personal Projects

 libDNS
libDNS is a C header library I am building for building and parsing DNS requests and responses. The guiding principles for it are:
  • Simple
  • Fast
  • No dynamic allocation
The current goal is just to implement basic DNS resolving in accordance with RFC 1035 but I hope to extend is with DNS over HTTPS and DNSSEC in the future.
Language: C
Repo: dns-resolver
 Conway's Game of Life
Conyway's Game of Life is a cellular automaton with the following rules:
  • Any live cell with fewer than two live neighbours dies
  • Any live cell with two or three live neighbours lives
  • Any live cell with more than three live neightbours dies
  • Any dead cell with exactly three live neighbors becomes a live cell
This rule set can produce some interesting patterns. I modelled the rule set using C++ and created two viewing interfaces.
  1. Terminal view using the Ncurses library
  2. Graphical view using SFML
Language: C++
Repo: Game of Life
 GroceryBudget
This is an Android app to help you keep track of how much you're spending when doing your groceries. It allows you to list the items you want to buy during your shop. Then as you are shopping you can add their prices and tick them off. The app will maintain a running count of how much you've spent.
Language: Java
Repo: GroceryBudget
 Flask Web App
For an informatics class that I took at university, in a team of two we built a we application which displays our findings on the change in cost of living in Australia. We gathered the data from the Australian Bureau of Statistics then cleaned and displayed it using Python. The website built using Flask displays our conclusions from the data and evidence to support them. Additionally, it includes a pivot table to allow the user to gain their own insights from the data.
Language: Python
